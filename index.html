04 — proyek

Proyek nyata,
dampak yang terasa

Health IT · Radiologi

Integrasi DICOM ke Satu Sehat

Sistem pengiriman data studi radiologi DICOM dari SIMRS Khanza ke platform Satu Sehat Kemenkes secara otomatis — menggantikan proses input manual yang sebelumnya memakan waktu petugas setiap hari.

  • Token OAuth2 Satu Sehat diambil otomatis — petugas tidak perlu login ulang
  • Data ServiceRequest radiologi langsung terhubung ke database Khanza
  • Studi DICOM dikirim ke PACS router Orthanc tanpa langkah tambahan
  • Laporan status pengiriman tampil per permintaan — jelas mana yang berhasil
  • Bisa proses banyak permintaan sekaligus, error per item tidak menghentikan yang lain
BashPythonOrthanc MariaDBFHIR R4Satu Sehat API

Health IT · Antrian

Sistem Tracking Antrian Pasien

Modul antrian berbasis web untuk SIMRS Khanza — pasien bisa pantau nomor antrian mereka dari ponsel sendiri, tanpa perlu bolak-balik nanya ke meja pendaftaran.

  • Pasien cukup masukkan NIK dan tanggal periksa — antrian langsung muncul
  • Info poli, nama dokter, jam registrasi, dan status tampil lengkap
  • Layar display antrian untuk ruang tunggu — bisa dipasang di TV atau monitor
  • Tampilan responsif di HP tanpa perlu install aplikasi apapun
  • Terhubung langsung ke database Khanza — tidak perlu ubah sistem yang ada
