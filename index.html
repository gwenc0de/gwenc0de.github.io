SIMRS · RADIOLOGI

Integrasi DICOM ke Satu Sehat

Sistem pengiriman data ImagingStudy DICOM dari SIMRS Khanza ke platform Satu Sehat Kemenkes, lengkap dengan bridging ServiceRequest radiologi secara otomatis.

  • Ambil token OAuth2 Satu Sehat otomatis — tidak perlu login manual
  • Query database Khanza untuk ServiceRequest ID yang sudah terkirim
  • Kirim ImagingStudy & studi DICOM ke PACS router (Orthanc)
  • Antarmuka CLI interaktif dengan ringkasan status tiap permintaan
  • Support multi-permintaan sekaligus dengan penanganan error per item
BashPythonOrthancMariaDBFHIR R4Satu Sehat API
$ ./kirim_dicom.sh → [OK] 3/3 studi berhasil dikirim
SIMRS · ANTRIAN

Sistem Tracking Antrian Pasien

Modul custom untuk SIMRS Khanza — pasien bisa cek posisi antrian mereka sendiri lewat halaman web tanpa perlu nanya ke petugas.

  • Tracking real-time nomor antrian berdasarkan NIK + tanggal periksa
  • Tampil info poli, dokter, jam registrasi, dan status terkini
  • Display antrian untuk ruang tunggu (layar TV / monitor)
  • Antarmuka mobile-friendly tanpa perlu install aplikasi
  • Integrasi langsung ke database Khanza tanpa modifikasi core
